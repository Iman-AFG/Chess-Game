# Simple Chess Game

A simple chess game built in Java using Swing.

With help of youtuber RyiSnow
https://www.youtube.com/watch?v=jzCxywhTAUI&t=2837s

## Features
- Chess board rendering
- Piece movement
- Move validation
- GUI using JFrame

## Technologies
- Java
- Swing
- Eclipse

## How to Run
1. Clone the repository
2. Open in Eclipse
3. Run Main.java
